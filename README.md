# 🌌 IronHUD: Cinematic Portfolio OS

**IronHUD** is a high-fidelity, production-grade landing page and interactive HUD inspired by "Stark Industries" aesthetics. It transforms a traditional portfolio into a visually-dense, motion-heavy command center using advanced SVG visualizations, glassmorphism, and cinematic animations.

## ⚡ Core Features

### 🛠️ Interactive Command Center
- **Dynamic Nav Modals**: A centralized modal registry (Overview, Dashboard, Users, Institute, Developer) featuring high-intensity backdrop blurs and spring-loaded transitions.
- **Glassmorphism UI**: Deep-black translucent panels (`#0a0a0c`) with 40px saturation blurs and subtle white borders for a premium "floating" feel.

### 📊 Visual Data Engine
- **Live Analytics Dashboard**: Interactive area charts with yellow gradient fills, real-time timescale switching (24h, 7d, 30d), and dynamic sparklines.
- **System Topology Map**: Animated SVG network diagrams showing node connectivity and server health with pulsing core routers.

### 🌀 Advanced Developer Profile
- **Orbital Arc Reactor Stack**: A custom SVG visualization where the tech stack (Next.js, React, Python, etc.) orbits a pulsing central "OS" core.
- **Sleek Contact Nodes**: High-quality SVG icons with magnetic hover effects and glowing indicators.

### 🎭 Cinematic UX
- **Cinematic Reveal**: A dramatic entry animation that "boots up" the interface with scanned line effects and particle systems.
- **Bento-Box Layouts**: Organized, data-dense grids inspired by modern Apple and linear.app design languages.

---

## 🚀 Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: Custom SVG + Lucide
- **Components**: Radix UI + Custom Bento Abstractions

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/sameerdubey2507/ironman.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/ui/` - Contains the core HUD panels (`nav-dashboard.tsx`, `nav-analytics.tsx`, etc.)
- `src/components/sections/` - Major page segments like `CinematicReveal.tsx`.
- `src/components/providers/` - Context providers for modal states and themes.
- `public/` - High-quality assets and backgrounds.
